# Dev Journal - Week 01

## Goals
- Setup project folder directory structure
- Configure Bootstrap CDN and custom stylesheet system
- Build navbar navigation header skeleton and footer

## Work Completed
- Initialized README, CONTRIBUTING, ROADMAP files.
- Upgraded project environment targets to use Bootstrap 5.3.
- Created `src/css/style.css` defining custom color palette tokens and Outfit/Inter fonts.
- Built responsive navbar layout with search inputs and badges count holders.

## Problems Encountered & Solutions
- *Problem*: Mobile navbar menu toggler didn't expand.
- *Solution*: Upgraded target attributes from Bootstrap 4 `data-toggle`/`data-target` to Bootstrap 5 `data-bs-toggle`/`data-bs-target`.

## Lessons Learned
- Framework naming conventions shift across major versions. Keep standard docs open during migrations.
