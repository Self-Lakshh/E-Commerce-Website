# Changelog - CommerceFlow

All notable changes to the CommerceFlow storefront will be documented in this file.

## [1.0.0] - 2024-01-12
### Added
- Dynamic shimmer skeleton loaders in search feeds.
- Toast notifications for cart addition and removal feedback.
- Accessible aria labels and keyboard focus outlines.
- Browser test runner verifying calculation systems.

## [0.8.0] - 2023-12-23
### Added
- Multi-step checkout flow (address fields, card payment validation, review).
- Order generation storing transaction logs in localStorage.
- Dynamic delivery estimates by speed.

## [0.6.0] - 2023-12-09
### Added
- Login, registration, and password recovery forms.
- Session persistence retaining active user details on reload.
- Tester credential accounts for direct QA checks.

## [0.4.0] - 2023-12-02
### Added
- Cart page listing, quantity adjust triggers, and item trash removals.
- Promotion discount math (SAVE10, WELCOME5 coupon support).
- Persistent list syncing in localStorage.

## [0.2.0] - 2023-11-18
### Added
- Product engine querying catalog items by rating, price, brand, and category.
- Specifications table lists on product details pages.
- Real-time customer reviews forms appending feedback arrays.
